>>> modele = ArbreClassification(profondeur_max=12)

class ForetClassification(n_arbres=1, n_attributs_par_split=1, profondeur_max=None)#

Forêt aléatoire pour la classification.

Paramètres:

  • n_arbres (int (défaut 1)) – Nombre d’arbres à inclure dans la forêt

  • n_attributs_par_split (int (défaut 1)) – Nombre d’attributs à tirer au hasard pour chaque séparation dans les arbres à construire

  • profondeur_max (int ou None (défaut None)) – Profondeur maximale acceptable des arbres à construire. Si None, les arbres complets sont construits (jusqu’à obtenir des feuilles pures).

Exemple

>>> modele = ForetClassification(n_arbres=100,
                                 n_attributs_par_split=20,
                                 profondeur_max=12)

charger_cifar()#

Charge le jeu de données de classification d’images CIFAR10.

Ce jeu de données contient des images d’objets et la tâche de classification consiste à retrouver le type d’objet contenu dans l’image (10 classes possibles).

Renvoie:

  • X_train (numpy.ndarray (~= liste de liste)) – jeu de données d’apprentissage (variables explicatives)

  • y_train (liste d’entiers) – variable cible pour le jeu de données d’apprentissage (=classe)

  • X_test (numpy.ndarray (~= liste de liste)) – jeu de données de test (variables explicatives)

  • y_test (liste d’entiers) – variable cible pour le jeu de données de test (=classe)

Exemple

>>> X_app, X_test, y_app, y_test = charger_cifar()
